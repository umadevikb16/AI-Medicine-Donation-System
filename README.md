# AI-Enabled Medicine Donation and Redistribution System

## 📌 Description
This project is a web-based platform that connects donors and NGOs to redistribute unused medicines efficiently. It ensures transparency, reduces wastage, and improves access to essential medicines.

## 🚀 Features
- OCR-based expiry date detection using Tesseract
- QR code-based tracking system
- NGO location-based matching
- Secure donation verification
- Intelligent medicine allocation logic

## 🛠️ Tech Stack
- Backend: Python (Flask)
- Database: MongoDB
- Frontend: HTML, CSS, JavaScript
- OCR: Tesseract

## 📊 How It Works
1. Donor uploads medicine details or image
2. System extracts expiry using OCR
3. NGOs are matched based on location
4. QR code generated for tracking
5. Medicines are collected and verified

## ▶️ How to Run
1. Install required libraries  
2. Run Flask app  
3. Open in browser
